# Songboard — слова песен для Holyrics

Адрес сайта - https://cyber0david.github.io/holychords-parser

Одностраничный инструмент (чистый HTML/CSS/JS, без сборки и зависимостей),
который помогает подготовить текст песни с holychords.pro для показа в Holyrics:

1. **Поиск** — открывает `holychords.pro/search` в новой вкладке.
2. **Вставка** — вы вставляете скопированный со страницы текст (с аккордами) и ссылку на песню.
3. **Аккорды убираются автоматически**, текст раскладывается по блокам
   (Куплет/Припев/Bridge и т.д., либо по пустым строкам, если меток нет).
4. **Редактирование** — блоки можно переставлять, объединять, разбивать по позиции
   курсора, переименовывать, дублировать, удалять.
5. **Экспорт** — копирование в буфер обмена или скачивание `.txt`, который
   вставляется в редактор песни Holyrics (пустая строка = граница слайда).


### Свой домен (опционально)
Если хотите использовать собственный домен вместо `github.io`:
1. В том же разделе **Settings → Pages** впишите домен в поле **Custom domain**.
2. У вашего DNS-провайдера добавьте `CNAME`-запись, указывающую на
   `<ваш-логин>.github.io` (для поддомена) либо `A`-записи на IP GitHub Pages
   (для корневого домена — см. официальную документацию GitHub Pages).
3. Подождите обновления DNS и включите **Enforce HTTPS** в настройках Pages.

Это полностью статический хостинг от GitHub — никаких Vercel/Netlify/прочих
посредников не требуется.
